# Contributing to JobTailor

Thank you for your interest in contributing to JobTailor! This document provides guidelines and information for contributors.

## 🚀 Ways to Contribute

- **🐛 Bug Reports**: Found a bug? [Open an issue](https://github.com/GoodBurger03/job-tailor/issues) with details
- **💡 Feature Requests**: Have an idea? [Create an issue](https://github.com/GoodBurger03/job-tailor/issues) describing your suggestion
- **🔧 Code Contributions**: Submit a pull request with your improvements
- **📖 Documentation**: Help improve documentation and guides

## 🛠 Development Setup

1. **Fork and Clone**:
   ```bash
   git clone https://github.com/GoodBurger03/job-tailor.git
   cd job-tailor/job-tailor
   npm install
   ```

2. **Environment Setup**:
   ```bash
   cp .env.example .env
   # Add your API keys to .env
   ```

3. **Start Development**:
   ```bash
   npm run dev
   ```

## 📝 Pull Request Process

1. **Create a Branch**: `git checkout -b feature/your-feature-name`
2. **Make Changes**: Implement your feature or fix
3. **Test**: Ensure your changes work and don't break existing functionality
4. **Commit**: Use clear, descriptive commit messages
5. **Push**: Push your branch to your fork
6. **Pull Request**: Create a PR against the main branch

## 🎯 Guidelines

### Code Style
- Follow existing code patterns and conventions
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused

### Commit Messages
- Use present tense: "Add feature" not "Added feature"
- Be descriptive but concise
- Reference issues: "Fix #123: Handle edge case"

### Testing
- Test your changes thoroughly
- Ensure existing functionality still works
- Consider edge cases and error scenarios

## 📋 Issue Guidelines

When reporting bugs or requesting features:

- **Bug Reports**: Include steps to reproduce, expected vs actual behavior, browser/OS info
- **Feature Requests**: Describe the problem you're trying to solve and your proposed solution
- **Questions**: Check existing issues first, then create a new one with clear details

## 📞 Contact

Questions? Feel free to [open an issue](https://github.com/GoodBurger03/job-tailor/issues) or reach out!

Thank you for contributing to JobTailor! 🎉
